# Küpte Yansıma (Cube Reflection Game)

Bu proje HTML5, CSS3 ve Three.js kullanılarak yapılmış 3 boyutlu bir zeka oyunudur.

## Nasıl Çalıştırılır?

Bu proje tamamen statik bir web sitesidir.
- **GitHub Pages**: Projeyi GitHub'a yükleyip Ayarlar > Pages kısmından aktifleştirerek doğrudan yayınlayabilirsiniz.
- **Yerel (VS Code)**: Visual Studio Code "Live Server" eklentisi ile `index.html` dosyasını sağ tıklayıp "Open with Live Server" diyerek açabilirsiniz.

**Not:** `server.py` dosyası artık gerekmediği için kaldırılmıştır. GitHub Pages ve diğer statik sunucular projeyi sorunsuz çalıştırır.

## Oynanış
1. **Döndürme**: Farenin sol tuşuna basılı tutup sürükleyerek küpü çevirin.
2. **Boyama**: Küpün yan yüzeylerindeki küçük karelere tıklayarak boyayın.
3. **Amaç**: Üst yüzeydeki şeklin, kenarlardan yansımasını (simetrisini) yan yüzeylere oluşturun.
4. **Kontrol**: Bitirdiğinizde "KONTROL ET" butonuna basın.
